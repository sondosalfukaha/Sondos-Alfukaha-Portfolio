Clever Mind POB ICT

Project Overview

This is a fullstack company web application where users can browse services . The platform is built using modern frontend and backend technologies.

Admin Control Panel

The system includes a secure admin panel, accessible only to authorized users. Admins can:

  • Edit the web application content
  • Control promotional banners and homepage content

The admin panel is built using Bootstrap and server-side APIs.

Technologies Used

  • Frontend: HTML, CSS, Bootstrap, JavaScript
  • Backend: Pure PHP
  • Admin Panel UI: HTML, CSS, Bootstrap, JavaScript
