Project Overview

The Goat is a fully-featured multi-language e-commerce web application. I developed both the frontend and backend of this platform, ensuring a seamless user experience along with powerful admin functionalities.

Admin Panel & Features

The platform includes Admin Panel where authorized administrators can:

  • Manage products, categories, and inventory
  • Control pricing in multiple currencies (JOD, EUR, USD)
  • Track and manage customer orders
  • Oversee payment status and order fulfillment
  • Manage site settings, content, and languages

Multi-Language Support

The platform is fully multilingual. Users can switch between supported languages dynamically, and all interface text and content are localized for a smooth international experience.

Tech Stack

  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Backend:PHP Laravel
